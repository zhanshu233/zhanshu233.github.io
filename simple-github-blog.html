# 开始行动

  • 首先要有一个Github的账号（没有的请点这里注册）。
    全球最大的男性同性交友网站，不懂的请Google之。
  • 登陆Github后点击“+ New repository”创建一个命名为username.github.io的project，注意要勾选生成README 。


    username就是注册Github时的账户名
  • 用Github帐号登录Simple（点击这里登陆）。
    Simple是一款极简的静态博客生成器，基于Github Pages，静态页面，完全在线操作，不需要服务器，只需一个 Github 账号即可。基于“在线”和“静态”这两种特性，我们可以更加专注于Blog的内容而不是后台维护，这也是我选择这种方式搭建个人博客的原因之一。
  • 点击Initialize来初始化Blog，之后点击Go就可以开始写Markdown的Blog了。
    只需第一次搭建的时候点击Initialize，之后每次直接点击Go就可以开始写Blog了。
  • 点击New post可以新建一篇文章，点击New page可以新建一个导航，点击Save保存并生成静态页面，访问username.github.io这个网址就可以看生成的Blog了。
    注意Github pages貌似有缓存，也就是说会有一段时间的延迟，可能不能立即看到效果，不过等一等就好了。

# 功能完善

到这一步的时候已经可以说大功告成了，但总觉得少了点什么？评论区！因为是静态页面，没有用到数据库，所以评论功能需要用第三方插件来实现。这里我们用免费的Disqus来实现。

  • 去Disqus官网注册（请点这里搭乘官网直达车）。
  • 点击右上角的小齿轮弹出下拉菜单，选择Add Disqus To Site，点击Star Using Engage。
  • 记住你在下图红框位置填写的shortname，点击Finish registration。
  • 去Github你的username.github.io下找到main.json，在"disqus_shortname":后面的引号内填入刚刚的shortname并保存。

# 结尾

至此，一个比较完整的个人博客就搭建完成了，谢谢isnowfy分享的开源项目Simple。
